Truth Table Generator

This script generates a truth table given a logic expression. The script was written in 2008, when I was just learning about Javascript. I've always liked logic, and hope to write more similar things in the future. The operators that can be used in the input expression are given in a table below.

Operators

Conjunction 與 &
Disjunction 或 |
Exclusive disjunction 異或 ^
Conditional 條件 >
Reverse conditional 反條件 <
Biconditional 雙條件 &
Negation 非 !
True 永真 1
False 永假 0
